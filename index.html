Aula 7-estudo de CSS

A sigla CSS é cascading style sheetsé uma linguagem utilizada para estilizar Pagina de web Utilizando esta linguagen podemos definir :

  • Cor de fundo, texto e elemento
  • fonte e tamanho das letras
  • posicionamento e elementos dentro da Pagina

O css funciona em conjunto em conjunto com o HTML, enquanto o HTML diz oque parece, o CSS diz como aparecer.
